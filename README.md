RainbowMarriage
===============

A Rainbow plugin.


Commands:
---------

 * `/marry <Player>`
    
    Marry another player!
    
    Permission: `rainbow.marry`
    
 * `/marry info <Player>`
   
    Get some info about another player.
    
    Permission: `rainbow.marry`
    
 * `/divorce <Player>`
   
    :(
    
    Permission: `rainbow.divorce`
 
 * `/marry SelectChurch`
   
   Setup the church location.
   
   Ops only.
